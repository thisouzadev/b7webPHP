# b7webPHP

### Rodar o (XAMPP)

```Terminal
sudo /opt/lampp/manager-linux-x64.run
```

### Permitir editar, criar e deletar dentro da pasta htdocs

```Terminal
sudo chmod -R 777 htdocs
```
